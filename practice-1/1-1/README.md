# Задание 1-1: Системные вызовы

## Вариант 4

Создать файл `data.txt` с текстом (5+ строк), подсчитать строки, записать результат в `lines.txt`.

## Требования

*   Используются только системные вызовы: `open()`, `read()`, `write()`, `close()`
*   НЕ используются: `fopen()`, `fprintf()`, `fscanf()`
*   Обработка ошибок с помощью `perror()`

## Описание программы

Программа выполняет следующие действия:

*   Создает файл `data.txt` с текстом объемом более 5 строк
*   Открывает файл для чтения с помощью `open()`
*   Читает файл по частям с помощью `read()`
*   Подсчитывает количество символов новой строки (`\n`)
*   Записывает итоговое количество строк в файл `lines.txt` с помощью `write()`
