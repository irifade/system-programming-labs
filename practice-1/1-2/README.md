# Задание 1-2: Процессы fork/exec

## Вариант 4: Цепочка процессов A→B→C

### Описание задания
Создать цепочку процессов A→B→C. Каждый процесс выводит сообщение формата: "Я процесс [имя], PID: [pid], родитель: [ppid]". Родительские процессы ждут завершения своих дочерних процессов с помощью функции `wait()`.

### Инструкция по запуску

1.  **Компиляция:** Убедитесь, что файл с исходным кодом называется `task.c`. Выполните команду компиляции:
    ```bash
    gcc -Wall -Wextra -o task task.c
    ```

2.  **Запуск:** Запустите скомпилированный исполняемый файл:
    ```bash
    ./task
    ```

### Результаты
Ниже приведен фактический вывод вашей программы при запуске:

```
Я процесс A, PID: 503, родитель: 302
Я процесс B, PID: 504, родитель: 503
Я процесс C, PID: 505, родитель: 504
```
