# Задание 1-3: Файловая система и права

## Вариант 4

### Описание задания

Создать директорию 'testdir'. Скопировать 3 файла из /etc (например /etc/hosts, /etc/hostname, /etc/issue). Установить права 444 для всех файлов. Попробовать изменить (должна быть ошибка).

### Инструкция по запуску

1. Убедитесь, что у вас есть исполняемый файл скрипта `script.sh` в текущей директории.
2. Сделайте скрипт исполняемым, если это еще не сделано:
   ```bash
   chmod +x script.sh
   ```
3. Запустите скрипт:
   ```bash
   ./script.sh
   ```
