# Практическая работа 1, Задание 1-4: Диагностика процессов

## Описание задания

**Вариант 4:**
Необходимо написать скрипт, который использует утилиты **ps**, **lsof**, **kill**, и работает с файловой системой **/proc** для диагностики запущенного процесса.

Требуется выполнить следующие шаги:
1.  Запустить `tail -f /var/log/syslog &` в фоновом режиме.
2.  Найти PID запущенного процесса.
3.  Выполнить `lsof -p [PID]` для просмотра открытых файлов.
4.  Сохранить команды и результаты в файл `openfiles.txt`.
5.  Завершить процесс.

## Инструкция по запуску

1.  **Сохраните** предоставленный ранее скрипт в файл `script.sh` в директории `.../practice-1/1-4`.
2.  **Убедитесь**, что файл имеет формат конца строк Unix (LF). При использовании стандартного Блокнота Windows может потребоваться преобразование утилитой `dos2unix`:
    ```bash
    dos2unix script.sh
    ```
3.  **Сделайте** файл исполняемым, используя команду `chmod`:
    ```bash
    chmod +x script.sh
    ```
4.  **Запустите** скрипт из терминала Bash:
    ```bash
    ./script.sh
    ```

Все команды и их вывод будут автоматически записаны в файл `openfiles.txt`.
