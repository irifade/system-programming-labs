# Отчет по Заданию 2-2: Синхронизация с мьютексами

## Вариант 4

**Задание:**
Глобальная переменная `sum=0`. Создать 4 потока. Каждый поток `i` добавляет к `sum` сумму чисел от 1 до `i*100`. Использовать мьютекс (`pthread_mutex_t`) для синхронизации доступа к глобальной переменной.

---


### 1. Компиляция и запуск
Откройте терминал WSL и выполните следующую команду для компиляции программы и ее немедленного запуска:

```bash
gcc task.c -o task -lpthread && ./task
```
### Компиляция

Ключ `-lpthread` обязателен для линковки библиотеки работы с потоками.
