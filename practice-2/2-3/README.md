# Отчет по Заданию 2-3: Обработка сигналов

## Вариант 4

**Задание:**
Программа работает в бесконечном цикле. При первом SIGINT выводит 
количество секунд работы. При втором SIGINT завершается. Использовать  signal() или sigaction(), обработчик должен быть async-signal-safe (использовать write() вместо printf()), volatile sig_atomic_t

---


### 1. Компиляция и запуск
Откройте терминал WSL и выполните следующую команду для компиляции программы и ее запуска:

```bash
gcc -Wall task.c -o task
```

### Запуск
```bash
./task
```
---

## 2. Результаты работы программы

Ниже представлен текстовый вывод работы скомпилированной программы:

### 

Running... Press Ctrl+C to see time. Press again to exit.^C
Seconds passed: 7
^C
Exiting…

Программа запускается и начинает считать время от старта.
При первом нажатии Ctrl+C (SIGINT) вызывается обработчик, который выводит прошедшее время и инкрементирует счетчик.
При втором нажатии Ctrl+C счетчик достигает двух, и программа принудительно завершается через безопасную функцию _exit().
