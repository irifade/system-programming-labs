# Практическая работа 2-1: Многопоточность

**Вариант:** 4

## Задание
Создать 5 потоков. Каждый засыпает на случайное время (1-5 секунд). После пробуждения выводит "Поток [ID] проснулся". Главный поток выводит общее время работы всей программы.

## Требования
* Использование библиотеки `pthread`
* Функции: `pthread_create()`, `pthread_join()`, `pthread_exit()`

## Инструкция по запуску
1. Скомпилируйте программу:
   ```bash
   gcc -pthread -o task task.c
2. Запустите исполняемый файл:
```bash
./task
